# Історія 6 клас · Екстернат · ІІ семестр

Готовий проєкт: локальний запуск + GitHub Pages + Google Apps Script + Telegram.

## Файли
- `index.html`
- `style.css`
- `app.js`
- `Code.gs`
- `README.md`

## Що вже працює локально
- стартовий екран
- введення ПІБ і класу
- інструкція
- 48 завдань
- випадковий порядок
- таймер 40 хвилин
- перерва 5 хвилин після 24 завдань
- світла / темна тема
- загальний результат
- ГР1 / ГР2 / ГР3
- локальна перевірка без обов’язкового підключення GAS

## Як запустити локально
1. Розпакуйте архів.
2. Відкрийте `index.html` у браузері.
3. Пройдіть роботу до кінця.
4. Кнопка надсилання локально не ламає роботу: якщо `GAS_URL` не заданий, просто покаже повідомлення.

## Як підключити GAS і Telegram
1. Створіть Apps Script проєкт.
2. Вставте код із `Code.gs`.
3. У **Project Settings → Script Properties** додайте:
   - `BOT_TOKEN`
   - `CHAT_ID`
4. Розгорніть як **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
5. Скопіюйте URL `/exec`
6. У `app.js` вставте цей URL у `CONFIG.GAS_URL`

## Як опублікувати на GitHub Pages
1. Створіть репозиторій.
2. Завантажте `index.html`, `style.css`, `app.js`, `README.md`
3. Увімкніть GitHub Pages:
   - Settings → Pages
   - Deploy from a branch
   - main / root
4. Перевірте роботу за публічним URL

## Модель оцінювання
- Загальний результат: 48 завдань
- ГР1: 24 завдання
- ГР2: 24 завдання
- ГР3: процедурний індикатор сумлінного виконання

## ГР3 враховує
- проходження обов’язкової перерви
- достатню кількість відповідей
- кількість перемикань вкладки / blur
- підозріло короткий час виконання
